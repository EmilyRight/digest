Ключ
  • NPS приложения «Ключ» по итогам второго полугодия 2023 года показал прирост на 8% и составил 44% — это один из самых высоких показателей среди продуктов «Ростелеком».
  • Приложение значительно доработано:
    • на 83% уменьшены вылеты пользователей из учетной записи;
    • улучшено качество и скорость техподдержки;
    • вторая линия техподдержки переведена из ДЭФИР и «РИТ»;
    • ликвидирован накопленный массив инцидентов;
    • упрощен клиентский путь идентификации жителя по документам на квартиру;
    • синхронизированы данные на платформе «Ключ» с АСР МРФ, снижено число жалоб.
  • Теперь домофонные ключи «Ключа» можно приобрести и на Wildberries.
  • На YouTube-канале «Ростелекома» запущен отдельный плейлист для «Ключа».
